# React Native App
> Storage and play Video Application

## Build
**1. Clone Project**
```
> git clone https://github.com/xd3011/Storage-And-Play-Video-Application
```
**2. Install packet**
```
> npm install <-----> npm i
```
**3. Client**
```
> npx expo start
```
**4. Scan QR with SmartPhone**

### Note: Run with Android SmartPhone (IOS Not Support)
